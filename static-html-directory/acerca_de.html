Acerca de BOCHAZO

Esta página muestra un ejemplo de código HTML con títulos y listas.

Ejemplo de HTML

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Ejemplo de listas</title>
</head>
<body>
  <h1>Ejemplo de listas y títulos</h1>
  <h2>Lista ordenada</h2>
  <ol>
    <li>Primer elemento</li>
    <li>Segundo elemento</li>
    <li>Tercer elemento</li>
  </ol>
  <h2>Lista desordenada</h2>
  <ul>
    <li>Elemento uno</li>
    <li>Elemento dos</li>
    <li>Elemento tres</li>
  </ul>
</body>
</html>

Descripción

El código anterior utiliza:

  • Un título principal con <h1>
  • Subtítulos con <h2>
  • Una lista ordenada <ol>
  • Una lista desordenada <ul>
